# Hackbotics YSWS

This is a website for a robotics related unofficial YSWS where you ship a vex related project and get vex parts/team registration/competition grants. It is made with Vite + React (and react router is in there somewhere in case i need to add more pages).

## Develop
Make sure you have npm and node, and git clone the project to your computer. Open a terminal in the folder and run:
```
npm i
npm run dev
```
Open https://localhost:5173 and see the result.

## Build
Make sure you have npm and node, and git clone the project to your computer. Open a terminal in the folder and run:
```
npm i
npm run build
```
